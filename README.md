📖 Project Title

Events.com – Upcoming Events Webpage

📝 Project Description

This is a simple HTML webpage designed to display upcoming events that users have registered for.
The page contains:

Website title and introduction
Upcoming events section
Two event banners with images
Event descriptions and details
Contact information in the footer

The webpage uses basic HTML tags and inline CSS styling.

🚀 Features

✅ Black background with white text (simple theme)
✅ Center aligned layout using <center> tags
✅ Two event sections with images
✅ Clickable image link for the first event
✅ Myntra fashion store hyperlink
✅ Footer with contact details

🛠️ Technologies Used
HTML5
Inline CSS
